Goals of Taking This Course

Through taking this course, I hope to:

  • Strengthen my web dev skills
  • Learn more about impactful and accessible design
  • Explore my passion for making maps!

The extent of my mapping experience has been taking GEOG 7 and working on some maps for the Daily Bruin, but I've never been able to take a class or work on projects where I get to define the scope and impact that it should have. I hope that this course will be a good starting point for me to continue exploring how I can use the tools I've learned about in a meaningful way.
